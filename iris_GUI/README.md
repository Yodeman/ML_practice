A python desktop app for predicting types of iris flower given some measurements
like:
    1.sepal length(cm)
    2.sepal width(cm)
    3.petal length(cm)
    4.petal width(cm)
Flower class:
    - Iris-Setosa
    - Iris-Versicolour
    - Iris-Virginica

The data set used is the sci-kit learn iris data set.
